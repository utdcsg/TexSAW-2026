The dist is a text file that contains the DNA encoding for a .osz file, which is an osu! beatmap. Osu! is referenced by the challenge title, with "touch the ovals" being a play on "click the circles".

In case the ATGC and .dna file extension wasn't enough, "Randall Munroe file transfer method" references the xkcd author's book "how to", where he proposed the ultra-efficient file transfer method of attaching dna encoded files to migrating butterflies.

The DNA encoding works by representing the file as trinary. The scheme works the same as the example on wikipedia, just with different nucleotides meaning different things. This is hinted at by me spelling "try" as "tri" in the challenge description.

.osz files are just renamed zips, so you can unzip the file. We are interested in the .osu file, which is a form of text file.

At the bottom are the positions of the notes. The x and y coordinates of every note are the same, and that each coordinate is a letter of the flag encoded in octal.
